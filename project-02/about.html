From struggle to structure: my path in web dev

I started with a single landing page and a lot of questions. Over time, I learned how pages connect, how styles scale, and how to make a site feel cohesive. The bumps along the way taught me discipline—write clean code, test every link, and never skip accessibility.

What changed most is confidence. I can now plan multi‑page architecture, build unified styling with distinct navigation patterns, and ship responsive layouts that look good across devices. You’ve been with me through the iterations; this portfolio reflects that growth.
